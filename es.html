Política de Privacidad – Planova

Última actualización: 01.05.2026

Planova es una aplicación de planificación operativa diseñada para funcionar sin infraestructura de servidores propios. Los datos generados por la aplicación permanecen en el dispositivo del usuario, salvo en los casos necesarios para el funcionamiento de las funciones solicitadas.

1. Datos personales

Planova no requiere la creación de una cuenta y no recopila directamente datos personales identificativos como nombre, dirección de correo electrónico o contactos.

Las compras dentro de la aplicación son gestionadas exclusivamente por Apple App Store y Google Play Store según sus respectivas condiciones.

2. Uso de la ubicación del dispositivo

Planova puede solicitar acceso a la ubicación del dispositivo.

La ubicación se utiliza exclusivamente para proporcionar previsiones meteorológicas locales dentro de la aplicación.

Para obtener dichas previsiones, las coordenadas geográficas pueden ser transmitidas a servicios meteorológicos externos.

La ubicación no se utiliza para fines de seguimiento, elaboración de perfiles o publicidad.

3. Localización de actividades

Las actividades planificadas pueden estar asociadas a una ubicación geográfica.

Las coordenadas utilizadas por la aplicación derivan de:

  • selección mediante servicios de mapas
  • procesos de geocodificación o geocodificación inversa

Esta información se utiliza exclusivamente para determinar el contexto operativo de las actividades y para el análisis de las condiciones ambientales.

4. Servicios externos

Para el funcionamiento de sus funciones, Planova interactúa directamente con servicios externos, entre ellos:

  • servicios meteorológicos, para obtener datos de previsión
  • servicios de geocodificación y localización, para gestionar coordenadas

Las solicitudes a estos servicios pueden incluir coordenadas geográficas o información necesaria para resolver ubicaciones.

5. Acceso al calendario

Planova puede solicitar autorización para acceder a los calendarios presentes en el dispositivo.

Si se concede la autorización, la aplicación puede crear y leer eventos para representar las actividades planificadas.

La gestión, sincronización y compartición de los calendarios se realiza exclusivamente a través de los servicios elegidos por el usuario y es responsabilidad del mismo.

6. Almacenamiento de datos

Todos los datos de la aplicación, incluidas las actividades y configuraciones, se almacenan exclusivamente en el dispositivo del usuario.

Planova no utiliza servidores propios para almacenar o procesar dichos datos.

7. Transferencia de datos

Planova no vende ni comparte datos personales.

Los datos pueden transmitirse exclusivamente en los siguientes casos:

  • solicitud de datos meteorológicos
  • uso de servicios de geocodificación
  • acceso a calendarios del dispositivo
  • exportaciones iniciadas por el usuario

8. Servicios de terceros

Planova puede interactuar:

  • directamente con servicios meteorológicos y de localización
  • indirectamente con servicios de calendario y sistemas de pago (Apple / Google)

Estos servicios operan según sus propias condiciones y políticas de privacidad.

9. Control del usuario

El usuario puede en cualquier momento:

  • desactivar el acceso a la ubicación desde la configuración del dispositivo
  • gestionar los permisos de calendario desde el sistema operativo
  • eliminar los datos desinstalando la aplicación

10. Contacto

Para cuestiones relacionadas con la privacidad:
