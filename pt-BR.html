Política de Privacidade – SkyPlan World Weather

Última atualização: 01.05.2026

SkyPlan World Weather é um aplicativo de planejamento operacional projetado para funcionar sem infraestrutura de servidores próprios. Os dados gerados pelo aplicativo permanecem no dispositivo do usuário, exceto quando necessário para o funcionamento das funcionalidades solicitadas.

1. Dados pessoais

O SkyPlan World Weather não exige criação de conta e não coleta diretamente dados pessoais identificáveis, como nome, endereço de e-mail ou contatos.

Qualquer compra dentro do aplicativo é processada exclusivamente pela Apple App Store e Google Play Store, de acordo com seus respectivos termos.

2. Uso da localização do dispositivo

O SkyPlan World Weather pode solicitar acesso à localização do dispositivo.

A localização é utilizada exclusivamente para fornecer previsões meteorológicas locais dentro do aplicativo.

Para obter essas previsões, as coordenadas geográficas podem ser transmitidas a serviços meteorológicos externos.

A localização não é utilizada para rastreamento, criação de perfis ou publicidade.

3. Dados de localização das atividades

As atividades planejadas podem estar associadas a uma localização geográfica.

As coordenadas utilizadas pelo aplicativo derivam de:

  • seleção através de serviços de mapas
  • processos de geocodificação ou geocodificação reversa

Essas informações são utilizadas exclusivamente para determinar o contexto operacional das atividades e processar condições ambientais.

4. Serviços externos

Para o funcionamento de suas funcionalidades, o Planova interage diretamente com serviços externos, incluindo:

  • serviços meteorológicos, para obter dados de previsão
  • serviços de geocodificação e localização, para gerenciar coordenadas

As solicitações a esses serviços podem incluir coordenadas geográficas ou informações necessárias para resolver localizações.

5. Acesso ao calendário

O SkyPlan World Weather pode solicitar autorização para acessar os calendários disponíveis no dispositivo.

Se autorizado, o aplicativo pode criar e ler eventos para representar atividades planejadas.

A gestão, sincronização e compartilhamento do calendário ocorrem exclusivamente através dos serviços escolhidos pelo usuário e permanecem sob sua responsabilidade.

6. Armazenamento de dados

Todos os dados do aplicativo, incluindo atividades e configurações, são armazenados exclusivamente no dispositivo do usuário.

O SkyPlan World Weather não utiliza servidores próprios para armazenar ou processar esses dados.

7. Transferência de dados

O SkyPlan World Weather não vende nem compartilha dados pessoais.

Os dados podem ser transmitidos exclusivamente nos seguintes casos:

  • solicitação de dados meteorológicos
  • uso de serviços de geocodificação
  • acesso aos calendários do dispositivo
  • exportações iniciadas pelo usuário

8. Serviços de terceiros

O SkyPlan World Weather pode interagir:

  • diretamente com serviços meteorológicos e de localização
  • indiretamente com serviços de calendário e sistemas de pagamento (Apple / Google)

Esses serviços operam de acordo com seus próprios termos e políticas de privacidade.

9. Controle do usuário

O usuário pode a qualquer momento:

  • desativar o acesso à localização nas configurações do dispositivo
  • gerenciar permissões de calendário através do sistema operacional
  • excluir os dados do aplicativo desinstalando-o

10. Contato
